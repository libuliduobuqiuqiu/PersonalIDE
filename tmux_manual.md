## tmux

移动pane:
```
<ctrl-a>/<ctrl-b>  h/j/k/l
```
设置pane宽度和高度:
```
<ctrl-a>/<ctrl-b>  H/J/K/L
```
将当前的pane最大化并且新建一个Window:
```
<ctrl-a>/<ctrl-b> + 
```
退出保存当前会话（detach):
```
<ctrl-a>/<ctrl-b> d
```
连接指定的会话（attach）:
```
<ctrl-a>/<ctrl-b> attach -t <session-name>
```
新建一个Window:
```
<ctrl-a>/<ctrl-b> !
```
窗口最大化（恢复窗口）:
```
<ctrl-a>/<ctrl-b> z
```
切换Window(指定编号):
```
<ctrl-a>/<ctrl-b> <Number>
```
